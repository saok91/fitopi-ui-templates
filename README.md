نیازمندی‌ها: Node.js

نصب وابستگی‌ها:

```
npm install
```

اجرا:

```
npm start
```

سرور روی آدرس http://localhost:4000 بالا می‌آید. صفحهٔ اصلی لیست طرح‌هاست؛ هر طرح از آدرس `/1` تا `/25` در دسترس است (مثلاً `/3/` برای design3، `/3/log.html` و `/3/chat.html`).
