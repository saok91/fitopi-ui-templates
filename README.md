# frontent-design-test

پروژهٔ نمایش چند طرح UI برای اپ سلامت (Fitopi). هر طرح شامل صفحهٔ اصلی، لاگ و چت است.

| شماره | طرح | طراح | ویژگی‌ها |
|-------|-----|------|----------|
| ۱ | Minimalist & Refined | Auto | مینیمال، نوار پیشرفت و چارت خطی ساده |
| ۲ | Maximalist & Playful | Auto | پرجزئیات، انیمیشن و افکت نئون، ایموجی |
| ۳ | Organic & Natural | Auto | تم گرم و ارگانیک، حلقهٔ پیشرفت کالری |
| ۴ | Brutalist & Raw | Auto | خام، فونت مونواسپیس، چارت میله‌ای |
| ۵ | Editorial & Magazine | Auto | شمایل مجله، شماره‌گذاری سکشن‌ها |
| ۶ | Organic Wellness | Opus 4.6 | تم سلامتی و طبیعت، نمودار با خط سقف مجاز |
| ۷ | Neon Vitality | Opus 4.6 | تم نئون و تکنو، لیبل‌های انگلیسی، استایل کد |
| ۸ | Soft Cloud | Opus 4.6 | پاستل و نرم، بخش تغییرات وزن و لژند |
| ۹ | Art Deco | Opus 4.6 | خطوط تزئینی آرت‌دکو، لژند نمودار |
| ۱۰ | Editorial Luxury | Opus 4.6 | ادیتوریل لوکس با خط طلایی |
| ۱۱ | Claymorphism | Gemini 3 Pro | کارت‌های نئومورفیک/کلی، سایه‌های نرم |
| ۱۲ | Holographic | Gemini 3 Pro | تم تیره، کارت شیشه‌ای، نئون سیان و صورتی |
| ۱۳ | Neobrutalism | Gemini 3 Pro | پس‌زمینه زرد، بوردر و سایه سخت |

نیازمندی‌ها: Node.js

نصب وابستگی‌ها:

```
npm install
```

اجرا:

```
npm start
```

سرور روی آدرس http://localhost:4000 بالا می‌آید. صفحهٔ اصلی لیست طرح‌هاست؛ هر طرح از آدرس `/1` تا `/15` در دسترس است (مثلاً `/3/` برای design3، `/3/log.html` و `/3/chat.html`).

مستندات بیشتر در پوشهٔ docs.
